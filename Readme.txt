Budget Tracker App Server

This server already deployed in the heroku and use the superbase postgres sql database.

When you want to run this server locally.
1. clone the repo.
2. run "npm install" command to install the dependency.
3. change the BaseURL in service> httpservice file.
4. run "npm run start" command to run the server.

When you want to change database.
1. go to src > config > config.json
2. chnage the database configuration from your configuration.
